# Developer Portfolio — Stanish Appalo JA

A clean, premium personal-brand site that showcases five web-development samples in one
place. Built with **vanilla HTML, CSS, and JavaScript**.

> This is the umbrella site to link from an Upwork profile or proposals — it ties together
> the five project samples and presents skills and contact info.

## ✨ Features

- Modern, responsive personal-brand design with an "available for work" status badge.
- A project grid where **each card carries its own gradient identity**, with live-demo and
  source links.
- About + skills (front-end, back-end, workflow) and a strong contact call-to-action.
- Mobile nav, sticky header, smooth scrolling — no frameworks.

## 🔧 Before publishing — update these

- **Contact email** in `index.html` (search for `your.email@example.com`).
- **Live-demo links** currently point to the sibling sample folders (great for local viewing).
  After you deploy each sample, swap them for the live URLs.
- **Source links** point to `https://github.com/stanish-appalo` — change to each repo's URL.

## 🚀 Run locally

```bash
python -m http.server 5500
# visit http://localhost:5500
```

(Open from the repo root so the relative `../` demo links resolve.)
